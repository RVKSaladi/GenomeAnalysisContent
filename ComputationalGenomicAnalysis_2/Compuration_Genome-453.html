Each COG corresponds to a particular function represented in a collection

of genomes. There are currently 3300 COGs. Some functions may appear in
all genomes, whereas others have a more restricted distribution. For example,
small ribosomal subunit protein S3 appears in all 43 reference genomes, but
ribonuclease I appears in only three. COGs have been organized into the
functional categories listed in Table 14.4.
