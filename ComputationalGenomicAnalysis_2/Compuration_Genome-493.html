Analysis:

–

The character (non-numeric) items are denoted by the quotation marks.

–

c(...,...,...)

concatenates or combines the elements in parentheses.

–

Note: The

+

in front of

"Smith J"

indicates that the command is wrapping

onto the next line. Do NOT include the

+

when typing

R

commands.

–

We can extract any element from this matrix by designating either the
dimension names or the row and column numbers: both

grades[4,3]

and

grades["Smith J","HW3"]

return the same value: 10.
