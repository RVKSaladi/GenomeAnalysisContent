We use the supplied

R

function

rnorm()

. The ﬁrst argument is the number

of data points to generate, and the second and third arguments set the mean
and standard deviation, respectively. The result is shown in Fig. A.1B. We can
control the appearance of the histogram by including additional arguments.
Often we want to alter the number of classes (

nclass

) or the range of the

plot, as in:

> hist(x.num,nclass=10,xlim=c(-10,10))

barplot

is useful for plotting from categorical data.

matplot

and

matlines

are useful for plotting rows and columns of a matrix in a single plot.

boxplot

is also useful for summarizing data. We can plot multiple ﬁgures (of any
type supported by

R

) on a single display by setting the parameters

mfrow

or

mfcol

.

mfrow

places the ﬁgures on the display sequentially by row, and obvi-

ously

mfcol

does the same thing but by column. Data plotted in the graphics

window can be printed or saved in a number of diﬀerent formats, such as

PostScript

,

PDF

,

JPEG

, or

PNG

. For example, we can select “Print” from the

“File” pull-down menu and then choose “Virtual Printer” and “PostScript
Settings” in the pop-up window to write a

PostScript

ﬁle to the chosen desti-

nation folder or directory.
