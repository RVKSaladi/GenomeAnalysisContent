Example: Calculating base composition

The following function calculates the base composition of a DNA sequence
string (represented numerically as

A->1, C->2

, etc.); i.e., it returns the pro-

portion of each type of base. It employs the supplied function

length()

that

we have discussed plus the ability to perform logical comparisons (in this case,

==

means “is equal to”) on vector elements, as described in Section A.4.

> basecomp<-function(inseq)
#Calculate base composition of an input sequence
#inseq is a vector of integers
{
f<-rep(0,4)

#vector to store base counts

for(i in 1:4)

{
f[i]<-length(inseq[inseq==i])/length(inseq)
}

return(f)
}
