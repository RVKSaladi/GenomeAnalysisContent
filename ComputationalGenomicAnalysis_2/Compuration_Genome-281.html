10.3 Similarity and Distance

Similarity and distance are measures of how closely- or distantly-related ob-
jects are based upon a collection of characters that describe those objects.
To illustrate these concepts, we consider an example with which we are all
familiar: relationships among some common animals. Each animal type is an
OTU. Consider the set of OTUs shown below. They have been scored for
dichotomous characters (1 = present, 0 = absent).
